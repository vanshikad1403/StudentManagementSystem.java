# Student Record Management System (Java)

This is a console-based Student Record Management System built in Java.  
It demonstrates the use of **OOP concepts**, **Interfaces**, **Inheritance**,  
**Exception Handling**, **File Handling**, **Multithreading**, and  
**Java Collections Framework**.

---

## Features
- Add, update, delete, and search student records  
- Persistent storage using `students.txt`  
- Sorting students by marks (descending)  
- Uses `List`, `Map`, `Iterator`, and `Comparator`  
- Custom exception: `StudentNotFoundException`  
- Multithreading used to simulate loading  
- Completely modular OOP structure  

---

## Technologies Used
- Java (JDK 8 or above)
- Collections Framework
- BufferedReader / BufferedWriter (File Handling)
- Multithreading (Runnable, Thread)
- Custom exceptions
